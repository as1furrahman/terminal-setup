# Terminal Setup

A cross-distro terminal environment for Arch Linux, Fedora, and Debian/Ubuntu.

## Quick Start

```bash
git clone https://github.com/as1furrahman/terminal-setup.git
cd terminal-setup
./install.sh
```

## Components

| Category | Tool |
|----------|------|
| Terminal | Ghostty |
| Shell | Zsh + Zinit |
| Prompt | Starship |
| Editor | Neovim (LazyVim) |
| File Manager | Yazi |
| Monitor | btop |

## Requirements

- Linux (Arch, Fedora, or Debian-based)
- Git
- sudo access
